# 🛰️ CX-PRO: Luxury Gold Administrative Ecosystem

A ultra-high-fidelity administrative dashboard system built with **Vue 3**, **TypeScript**, and **Vite**, featuring the exclusive **Luxury Gold (CX-PRO)** design language.

## ✨ Core Design Philosophy

The CX-PRO design system is engineered for elite management environments, balancing deep atmospheric aesthetics with high-performance utility.

- **Vibrant Gold Palette**: A multi-stage gold system ranging from rich ambers to brilliant white-gold.
- **Atmospheric Luminance**: Deep radiant glows, additive blending, and "Golden Dawn" light sources.
- **Glassmorphic Precision**: High-blur surfaces with sharp, vibrant borders.
- **Dynamic Interaction**: Icon-rail navigation with secondary detail panels and wavy fluid footer visuals.

## 🚀 Key Features

- **Icon Rail Sidebar**: A space-efficient, fixed navigation rail featuring Micro-labels and high-intensity glowing orbs.
- **Global Settlement Lattice**: A high-fidelity data visualization centerpiece for real-time density mapping.
- **Protocol Journal**: Sophisticated intelligence logging system with status indicators.
- **Modular Architecture**: Clean separation between common UI components (`CxCard`, `CxButton`, `CxSidebar`, `CxToolbar`) and business views.
- **Responsive Workspace**: Master-level administrative experience optimized for high-resolution displays.

## 🛠️ Technology Stack

- **Frontend**: Vue 3 (Composition API + Script Setup)
- **Styling**: TailwindCSS + Custom Vanila CSS (CX Design Tokens)
- **Icons**: Lucide Vue Next (Customized CX variants)
- **Build Tool**: Vite 
- **Language**: TypeScript

## 📁 Project Structure

- `src/components/common/cx/`: The core CX-PRO component library.
- `src/views/manager/admin/`: The primary administrative dashboard modules.
- `src/style.css`: Centralized luxury design tokens and utility classes.

## 📦 Installation & Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
